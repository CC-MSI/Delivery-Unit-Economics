# Delivery Unit Economics Model

Interaktives Modell zur Analyse der Profitabilität von Lebensmittellieferungen im deutschen Markt.

## Features
- Berechnung von EBIT pro Bestellung
- Simulation verschiedener Szenarien (Warenkorb, Logistik, Kosten)
- Vergleich von Fulfillment- und Liefermodellen
- Visualisierung der Kostenstruktur

## Nutzung
Einfach `index.html` im Browser öffnen.

## Deployment (GitHub Pages)
1. Repository erstellen
2. Dateien hochladen
3. In Settings → Pages aktivieren (Branch: main / root)

## Hinweis
Modell basiert auf Marktannahmen und dient zur Simulation.
